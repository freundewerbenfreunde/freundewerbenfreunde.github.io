§ 3

Angebote

Über sein Konto kann der Benutzer Angebote erstellen.

Jedes Angebot wird an drei Stellen angezeigt:

  • Auf der Angebotsseite jeweiligen App oder Webseite
  • Auf der Profilseite seines Verfassers
  • Auf der Angebotverwaltungsseite seines Verfassers

Ein Angebot besteht aus folgenden Informationen:

  • Titel
  • Beschreibung
  • Link: Empfehlungslink oder Link zur App oder Webseite
  • Code: falls der Code nicht im Empfehlungslink enthalten ist, optional
  • Gültigkeitsdatum: falls das Angebot zeitlich begrenzt ist, optional

Dabei ist folgendes zu beachten:

  • Das Angebot ist für die entsprechende App oder Webseite gültig.
  • Der Code darf nicht umgewandelt oder von seinem Original abgewandelt werden (z.B. Url-Verkürzer sind verboten).

  • Der Benutzer verpflichtet sich, über alle Rechte zu verfügen, die mit der Fähigkeit verbunden sind, das Angebot auf der Webseite zu teilen.

Der Benutzer kann mehrere Aktionen mit seinem Angebot durchführen:

  • Das Angebot bearbeiten
  • Das Angebot löschen

Um die Aktualität der Angebote sicherzustellen, werden Angebote, die seit 2 Monaten nicht aktualisiert worden sind automatisch gelöscht.

Der Benutzer kann in seinem Konto bestimmte Angebote löschen. Eine Wiederherstellung der gelöschten Daten ist nicht möglich.
